Using the PlayStation2 IPU Stream Exporter Plug-in for Adobe Premiere for Windows

Version 1.0
Mike Dean, Sony Computer Entertainment Europe
10th February 2000


Installing

To install the exporter plug-in, place the file ‘CMPM-PlayStation2 IPU Stream.prm’ into the appropriate plug-ins directory used by Adobe Premiere (such as 'C:\Program Files\Adobe\Premiere 5.1\Plug-ins').

Using

To invoke the exporter plug-in, select the File/Export/Movie item from the menu bar (the timeline should be the active window for the option to be available). In the file requestor which consequently opens, click on the 'Settings...' button.

In the 'General Settings' menu of the dialog box which then opens is a 'PlayStation2 IPU Stream' option which should be selected.

Select the 'work area' required, and ensure that the 'Open When Finished' checkbox is unticked.

When the General Settings dialog has been dismissed and a filename for the exported stream has been specified, a window will appear requesting details of compression settings. In the current version of the exporter, the only option available is a setting for the maximum size, in bytes, of compressed frame images.

Previewing

The files output by this exporter conform to the '.ipu' file format, and can be previewed using the 'ezmovie' application featured in the sample code of the development software.

Support

Please email ps2_support@scee.net to report bugs in the exporter, documentation errors, or to request additional functionality.

Known Bugs

  • Having clicked 'Save' in the file requestor, aborting the export (even before it has begun) will not erase the incomplete file.
  • While not a bug, it is acknowledged that the performance of the exporter could be better..

Future Plans

Aside from performance issues, no major improvements are currently planned for this exporter. If you can suggest any features which could be useful additions, please email the support address above.
